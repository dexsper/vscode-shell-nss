# Shell NSS Syntax Highlighting

A Visual Studio Code extension that provides syntax highlighting, autocomplete, and hover descriptions for [Shell](https://github.com/moudey/Shell) configuration files.

## Features

- **Syntax Highlighting**: Full syntax highlighting for Shell NSS configuration files
- **Autocomplete**: Intelligent code completion for keywords with proper snippet insertion
- **Hover Descriptions**: Detailed descriptions when hovering over keywords
- **Language Support**: Recognizes `.nss` file extension and provides language-specific features

## Requirements

- Visual Studio Code version 1.107.0 or higher

## Installation

1. Open Visual Studio Code
2. Go to Extensions view (Ctrl+Shift+X)
3. Search for "Shell NSS Syntax Highlighting"
4. Click Install
